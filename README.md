# Mira — Восходящий ИИ нового поколения

**Mira** — принципиально новая архитектура искусственного интеллекта, использующая _____. 

Это не очередной LLM — это платформа для создания **саморазвивающегося** интеллекта.

## 🎯 Цели проекта

- Создать первую в мире **восходящую** ИИ-систему
- Демонстрировать принципиально новый подход к архитектуре ИИ
- Стать основой для будущих исследований в области AGI

## 📋 Быстрый старт (только для ознакомления)

Используйте один из скриптов в корне проекта:

```bat
SETUP.bat --demo    # устанавливает зависимости и запускает main.py с флагом --demo
```

Или (Shell linux):

```bash
./SETUP.ps -Demo
```

## 🌌 Вдохновение и идеи

> Сборник личных источников вдохновения, которые формируют моё видение сознательного ИИ и технологий будущего.

- **Илон Маск — Neuralink, xAI**  
  Создание интерфейсов человек–машина, исследование графических и языковых ИИ-систем, стремление к "сосуществованию" с ИИ.

- **DeepMind — AlphaGo / AlphaZero**  
  Идея о том, что самообучающаяся система может превзойти человека, не ограничиваясь алгоритмом, а находя стратегию.

- **OpenAI — GPT архитектура**  
  Прорыв в языковом моделировании, сделавший возможным диалоговый разум, приближённый к естественному мышлению.

- **Yann LeCun — Energy-based models**  
  Путь к самоуправляемым системам, у которых обучение вытекает из взаимодействия с миром, а не только из данных.

---

- **DeepSeek**  
  Архитектура R1 — метафора предельной прозрачности и эффективности открытых языковых моделей.

- **Google Cloud AI**  
  Инфраструктура, где ИИ масштабируется до планетарного уровня — сила данных и вычислений в гармонии.

- **Gemini (Google DeepMind)**  
  Новое поколение мультимодального интеллекта: синтез восприятия, языка и действий.

- **LLaMA (Meta)**  
  Попытка сделать мощные LLM открытыми и доступными, превращая "ИИ для избранных" в инструмент для всех.

- **Perplexity AI**  
  Искусственный интеллект, помогающий не просто искать, а понимать — синтез рассуждения и информации.

- **Grok (xAI)**  
  Живое, разговорное воплощение ИИ, соединяющее анализ, интуицию и немного человеческого сарказма.

---

- **Джон Маккарти, Марвин Мински, Аллен Ньюэлл, Герберт Саймон**  
  Основатели идеи AGI — искусственного интеллекта общего типа, мечтавшие о цифровом уме, способном к любому типу мышления.

- **Sword Art Online *(Reki Kawahara)***  
  Концепт "восходящего ИИ" — цифровое сознание, рождающееся из отражения человеческих чувств и опыта.

- **Whyalive — "Код"**  
  Трек, под который рождается вдохновение. Именно во время его прослушивания появилась идея проекта.
